# Skills (opcional)

Este kit incluye una skill base (`microservices`) y *bridges* para skills externas.

- `microservices`: targets + evidencia + gates + wrap.
- `ui-ux-pro-max` (bridge): guía para usar una skill de UI/UX externa.
- `prompt-generator-suite` (bridge): guía para usar generadores de prompts.

Las skills externas se guardan como referencia para que el workflow sea reproducible.
